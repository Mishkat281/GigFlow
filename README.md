# GigFlow
GigFlow is a full-stack freelance marketplace web app where users can create gigs, place orders, chat in real time, and leave reviews. It is built with the MERN stack and includes secure authentication, order management, and messaging features.

## Tech Stack
Frontend: React, Vite
Backend: Node.js, Express, MongoDB

## Setup
Create a `.env` file in backend:
MONGO=your_mongodb_uri
JWT_KEY=your_secret

Run backend with `npm start`
Run frontend with `npm run dev`
